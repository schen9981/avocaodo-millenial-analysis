Project Overview and Hypothesis

This projects seeks to explore the trends in avocado consumption in the US, in connection with millenials, a group classified as those born between 1980 and 1994.

We expect that cities in the United States with a higher population of millenials, especially those with a high population of millenials with above-average income, consume more avocados despite prices increases in recent years.

Reorganization and Joining Data

Our analysis was dependent on common regions (California, Plains, South Central, Midsouth, Southeast, Great Lakes, Northeast) defined by the Hass Avocado Board for both the avocado consumption data and the demographic data.

Our raw data composed of the avocado consumption for 2016, 2017, and 2018 for cities/states specified in the raw data, as well as the accumulated consumption data for Hass Avocado Board defined regions. Thus, we extracted out the accumulated consumption for the regions specified above, into a new table.

As for our raw data for demographic, this had data regarding gender, age, and population for each state for each of the years 2016, 2017, and 2018. We first removed the data for Alaska and Hawaii, as these states were not taken into account for the avocado consumption regions. We created a new column called 'region', where we labeled each state with its appropriate region. Then, for each state in region X, we added up population for each age (while keeping gender separated).

This way, we have a 'region' parameter across our demographic and avocado consumption data that we can use to join the tables to perform analysis.
